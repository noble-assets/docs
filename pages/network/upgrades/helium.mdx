# Helium<sup>2</sup> Upgrade

### Highlights

- Upgrades Noble itself and all of it's core modules to the latest stable release of the Cosmos SDK, v0.50.x a.k.a. Eden.
- [FiatTokenFactory](https://github.com/circlefin/noble-fiattokenfactory) The BlockIBC logic was improved to support both Bech32 and Bech32m for IBC recipient addresses.
- [Florin](https://github.com/monerium/module-noble) The module was updated to accept a user's public key when verifying signatures, instead of relying on on-chain data.
- [Forwarding](https://github.com/noble-assets/forwarding) The module was updated to include a fallback address and a list of allowed denominations to forward.
